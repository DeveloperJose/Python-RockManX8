Getting Started
====================
I suggest you make a backup of your text files (located in the folder named mes) before using this editor.
You need to make sure the "mes" folder is NOT read-only before running this editor. 


Troubleshooting
====================
If you get an error that says
        PermissionError: [Errno 13] Permission denied: ''
then your MCB folder is read-only, please see the getting started section
